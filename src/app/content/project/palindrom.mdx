# Memahami Palindrom dengan Array

## Pendahuluan

Dalam pemrograman, **palindrom** adalah sebuah kata, frasa, angka, atau urutan karakter lainnya yang dapat dibaca sama baik dari depan maupun dari belakang (mengabaikan spasi, tanda baca, dan perbedaan huruf besar kecil). Contoh dari palindrom adalah "radar", "level", "12321".

Dalam konteks **array**, kita sering kali diminta untuk memeriksa apakah sebuah array atau string adalah palindrom. Berikut adalah penjelasan lengkap tentang cara memeriksa palindrom menggunakan array.

## Konsep Dasar

Untuk memeriksa apakah sebuah array adalah palindrom, kita harus membandingkan elemen-elemen
dari array tersebut dari dua arah: dari depan ke belakang dan dari belakang ke depan.
Jika semua elemen yang berlawanan sama, maka array tersebut adalah palindrom.

### Langkah-Langkah Umum

1. **Inisialisasi** dua indeks: satu di awal (misalnya `i`), dan satu di akhir (misalnya `j`).
2. **Bandingkan** elemen pada posisi `i` dan `j`.
3. Jika elemen tidak sama, maka array bukan palindrom.
4. Jika elemen sama, lanjutkan dengan menggeser `i` ke kanan dan `j` ke kiri.
5. Ulangi proses ini sampai `i` melewati `j`.

## Implementasi dalam Bahasa Pemrograman

Berikut ini adalah beberapa implementasi untuk memeriksa palindrom menggunakan array dalam berbagai bahasa pemrograman.

### Python

```python
def is_palindrome(arr):
    left = 0
    right = len(arr) - 1

    while left < right:
        if arr[left] != arr[right]:
            return False
        left += 1
        right -= 1

    return True

# Contoh penggunaan
arr = ['r', 'a', 'd', 'a', 'r']
print(is_palindrome(arr))  # Output: True
```

### Java

```java
public class PalindromeCheck {
    public static boolean isPalindrome(char[] arr) {
        int left = 0;
        int right = arr.length - 1;

        while (left < right) {
            if (arr[left] != arr[right]) {
                return false;
            }
            left++;
            right--;
        }

        return true;
    }

    public static void main(String[] args) {
        char[] arr = {'r', 'a', 'd', 'a', 'r'};
        System.out.println(isPalindrome(arr));  // Output: True
    }
}
```

## Penjelasan Tambahan

- **Kompleksitas Waktu**: Algoritma ini memiliki kompleksitas waktu O(n) karena hanya memerlukan satu kali iterasi melalui array.
- **Kompleksitas Ruang**: Algoritma ini memiliki kompleksitas ruang O(1) karena tidak memerlukan ruang tambahan yang signifikan selain beberapa variabel indeks.

### Pertimbangan Tambahan

1. **Kasus Sensitifitas Huruf**: Jika kita bekerja dengan string dan ingin mengabaikan perbedaan antara huruf besar dan kecil, kita perlu menambahkan langkah untuk mengubah semua karakter menjadi huruf kecil atau huruf besar sebelum melakukan perbandingan.
2. **Mengabaikan Spasi dan Tanda Baca**: Untuk palindrom yang mengabaikan spasi dan tanda baca, kita perlu membersihkan string dari karakter-karakter tersebut sebelum melakukan perbandingan.

#### Contoh dengan Mengabaikan Huruf Besar/Kecil dan Spasi

```python
import re

def is_palindrome(s):
    s = re.sub(r'[^A-Za-z0-9]', '', s).lower()
    left = 0
    right = len(s) - 1

    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1

    return True

# Contoh penggunaan
s = "A man, a plan, a canal, Panama"
print(is_palindrome(s))  # Output: True
```
