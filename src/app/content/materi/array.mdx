# Memahami Array dalam Pemrograman

## Pendahuluan

Array adalah salah satu struktur data yang paling fundamental dalam pemrograman. Array memungkinkan kita untuk menyimpan sekumpulan elemen yang memiliki tipe data yang sama dalam satu variabel. Dengan menggunakan array, kita bisa mengorganisasi dan mengelola data secara efisien.

## Pengertian Array

Array adalah kumpulan elemen yang diakses dengan menggunakan indeks. Indeks array biasanya dimulai dari 0, yang berarti elemen pertama dari array berada pada indeks 0, elemen kedua pada indeks 1, dan seterusnya.

### Ciri-Ciri Array

1. **Homogen**: Semua elemen dalam array harus memiliki tipe data yang sama.
2. **Indeks Terurut**: Setiap elemen dalam array diakses melalui indeks yang terurut.
3. **Ukuran Tetap**: Ukuran array biasanya ditentukan saat deklarasi dan tidak dapat diubah.

## Deklarasi dan Inisialisasi Array

### Bahasa Pemrograman Python

#### Deklarasi dan Inisialisasi

```python
# Deklarasi dan inisialisasi array (list) dalam Python
arr = [1, 2, 3, 4, 5]
```

#### Mengakses Elemen Array

```python
# Mengakses elemen array
print(arr[0])  # Output: 1
print(arr[2])  # Output: 3
```

#### Mengubah Elemen Array

```python
# Mengubah elemen array
arr[1] = 10
print(arr)  # Output: [1, 10, 3, 4, 5]
```

#### Deklarasi dan Inisialisasi

```java
// Deklarasi dan inisialisasi array dalam Java
int[] arr = {1, 2, 3, 4, 5};
```

#### Mengakses Elemen Array

```java
// Mengakses elemen array
```

#### Deklarasi dan Inisialisasi

```cpp
// Deklarasi dan inisialisasi array dalam C++

```

#### Mengakses Elemen Array

## Operasi Dasar pada Array

### Traversing (Melintasi) Array

Traversing adalah proses mengakses setiap elemen dalam array satu per satu. Contoh dalam Python:

```python
# Traversing array
for element in arr:
    print(element)
```

### Pencarian (Searching) dalam Array

#### Linear Search

Pencarian linear adalah metode pencarian elemen dengan memeriksa setiap elemen satu per satu.

```python
# Linear search dalam Python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

arr = [1, 2, 3, 4, 5]
print(linear_search(arr, 3))  # Output: 2
```

### Pengurutan (Sorting) Array

#### Bubble Sort

Bubble Sort adalah algoritma sederhana untuk mengurutkan elemen dalam array dengan membandingkan setiap elemen dengan elemen di sebelahnya dan menukarnya jika urutannya salah.

```python
# Bubble sort dalam Python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print("Sorted array is:", arr)
```

### Menambah dan Menghapus Elemen

#### Menambah Elemen

Menambah elemen biasanya dilakukan dengan menciptakan array baru dengan ukuran lebih besar dan menyalin elemen-elemen dari array lama ke array baru.

#### Menghapus Elemen

Menghapus elemen dari array melibatkan pergeseran elemen-elemen lain untuk mengisi tempat yang kosong.

### Python

```python
# Menambah elemen (menggunakan list karena array Python bersifat dinamis)
arr.append(6)
print(arr)  # Output: [1, 10, 3, 4, 5, 6]

# Menghapus elemen
arr.remove(10)
print(arr)  # Output: [1, 3, 4, 5, 6]
```

## Array Multidimensi

Array multidimensi adalah array yang memiliki lebih dari satu indeks. Contoh yang paling umum adalah array dua dimensi (matriks).

### Bahasa Pemrograman Python

```python
# Array dua dimensi dalam Python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Mengakses elemen
print(matrix[0][1])  # Output: 2

# Mengubah elemen
matrix[1][1] = 10
print(matrix)  # Output: [[1, 2, 3], [4, 10, 6], [7, 8, 9]]
```

### Bahasa Pemrograman Java

```java
// Array dua dimensi dalam Java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Mengakses elemen
System.out.println(matrix[0][1]);  // Output: 2

// Mengubah elemen
matrix[1][1] = 10;
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}
```

### Bahasa Pemrograman C++

```cpp
// Array dua dimensi dalam C++
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Mengakses elemen
cout << matrix[0][1] << endl;  // Output: 2

// Mengubah elemen
matrix[1][1] = 10;
for(int i = 0; i < 3; i++) {
    for(int j = 0; j < 3; j++) {
        cout << matrix[i][j] << " ";
    }
    cout << endl;
}
```

## Kesimpulan

Array adalah struktur data penting yang memungkinkan kita menyimpan dan mengelola sekumpulan elemen yang homogen secara efisien. Dengan memahami cara mendeklarasikan, menginisialisasi, mengakses, dan memodifikasi array, serta operasi dasar seperti pencarian dan pengurutan, kita dapat memanfaatkan array untuk berbagai kebutuhan pemrograman.

Pemahaman array multidimensi juga penting untuk bekerja dengan data yang lebih kompleks, seperti matriks. Dengan menguasai konsep-konsep ini, kita dapat menangani banyak masalah pemrograman dengan lebih efektif dan efisien.
