<h2 class="text-primary my-2"> Untuk Guru </h2>

<h3>Login/Sign Up</h3>
1. Masuk ke Halaman Utama dengan link <a href="https://projectful.vercel.app" class="hover:underline underline-offset-2 font-bold text-primary">projectful.vercel.app</a> atau <a href="https://projectful.netlify.app" class="hover:underline underline-offset-2 font-bold text-primary">projectful.netlify.app</a>, kemudian klik tombol <mark class="bg-primary text-primary-foreground">Mulai</mark>.
2. Pada Halaman Login, silakan login menggunakan akun masing-masing
3. Jika belum memiliki akun, silakan masuk ke Halaman Daftar disini.
4. Pada Halaman Sign Up, isikan data diri sesuai dengan ketentuan yang ada pada kolom input masing-masing. Kemudian klik tombol <mark class="bg-primary text-primary-foreground">Submit</mark>. Halaman akan *redirect* ke Halaman Login

<br/>
<h3>Dashboard</h3>
Setelah Anda masuk ke halaman Dashboard, Anda dapat menilai project siswa selesai dengan mengubah isi di kolom Status dengan **FINISHED**

<br/>
<h3>Tambah Project</h3>
1. Anda dapat menambahkan Project dengan mengklik tombol <mark class="bg-primary text-primary-foreground">+</mark> di samping Tambah Project.
2. Pada Halaman Tambah Project, Anda bisa mengisikan Judul Project, Deskripsi Project, dan Tenggat Waktu Project. Setelah selesai, Anda bisa mengklik <mark class="bg-primary text-primary-foreground">Buat Project</mark>.

<br/>
<h3>Tambah Assessment</h3>
1. Anda dapat menambahkan Assessment dengan mengklik tombol <mark class="bg-primary text-primary-foreground">+</mark> di samping Tambah Assessment.
2. Pada Halaman Tambah Assessment, Anda bisa mengisikan Judul Assessment, Deskripsi Assessment, Pertanyaan Soal, dan Opsi Jawaban.
3. Anda dapat menambah atau mengurangi Pertanyaan Soal dengan mengklik <mark class="bg-primary text-primary-foreground">Tambah Soal</mark> dan <mark class="bg-primary text-primary-foreground">Hapus Soal</mark>.
4. Anda juga dapat menambah atau mengurangi Opsi Jawaban dengan mengklik <mark class="bg-primary text-primary-foreground">Tambah Opsi</mark> dan <mark class="bg-primary text-primary-foreground">Hapus Opsi</mark>.
5. Setelah selesai, Anda bisa mengklik <mark class="bg-primary text-primary-foreground">Buat Assessment</mark>.

<br/>
<h3>Tambah Materi</h3>
1. Anda dapat menambahkan Materi dengan mengklik tombol <mark class="bg-primary text-primary-foreground">+</mark> di samping Tambah Materi.
2. Pada Halaman Tambah Materi, Anda bisa mengisikan Judul Materi, Deskripsi Materi, dan Isi Materinya. Setelah selesai, Anda bisa mengklik <mark class="bg-primary text-primary-foreground">Buat Materi</mark>.
